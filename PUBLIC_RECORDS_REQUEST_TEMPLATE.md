## Request Summary

Pursuant to Florida's Public Records Law (Chapter 119, Florida Statutes), the Marion County Surtax Oversight Committee respectfully requests access to the following records related to school surtax-funded capital improvement projects:

---

## Records Requested

### 1. Project Phase Documentation
For each active surtax project, we request:
- Detailed project phase schedules showing start and end dates
- Current status of each project phase
- Percent completion for each phase
- Any phase delay documentation or explanations

**Projects Requiring This Information**: All 44 active surtax projects

---

### 2. Inspection Reports and Quality Assurance Records
We request all inspection documentation including:
- **Inspection Reports**: Complete inspection reports with dates, inspector names, and findings
- **Deficiency Logs**: Documentation of any deficiencies identified and remediation actions
- **Quality Assurance Tests**: Results of any quality tests or certifications
- **Compliance Documentation**: Code compliance verification and certificates of occupancy

**Specific Need**: We currently have limited inspection records and need comprehensive documentation to ensure proper oversight of construction quality and safety compliance.

---

### 3. Community Engagement Records
We request documentation of all community engagement activities:
- **Public Meeting Records**: Dates, attendance logs, agendas, and minutes
- **Community Feedback**: Summaries of feedback received from community members
- **Concerns Log**: Documentation of concerns raised by community and responses provided
- **Public Presentations**: Copies of presentations given at community meetings

**Purpose**: To track public involvement and ensure transparency in the use of taxpayer funds.

---

### 4. Committee Action Items and Meeting Documentation
We request:
- **Committee Meeting Minutes**: All meeting minutes from surtax oversight committee meetings
- **Action Item Tracking**: Documentation of action items, assignments, and completion status
- **Decision Records**: Formal votes and decisions made by the committee
- **Follow-up Documentation**: Evidence of completed action items

**Time Period**: All records from the inception of the surtax program to present

---

### 5. Contractor Performance Records
For each contractor working on surtax projects:
- **Performance Evaluations**: Quality scores and performance reviews
- **Safety Records**: OSHA reports, safety incidents, lost-time accidents
- **Past Project History**: List of past projects completed with the district
- **Deficiency Rates**: Historical deficiency rates and corrective actions
- **Local Hiring Data**: Documentation of local hiring percentages and workforce demographics

**Purpose**: To evaluate contractor performance and ensure accountability.

---

### 6. Detailed Financial Records
Beyond basic budget information, we request:
- **Funding Source Breakdown**: Detailed breakdown of funding sources for each project
- **Budget Amendments**: All approved budget amendments with justifications
- **Expenditure Details**: Line-item expenditures for each project
- **Contingency Tracking**: Contingency fund usage and remaining balances
- **Cost Per Square Foot Analysis**: Calculations and comparative benchmarks

---

### 7. Project Justification Documentation
For each project, we request:
- **Needs Assessment**: Original needs assessment or justification for the project
- **Scope of Work Documents**: Detailed scope statements and specifications
- **Community Impact Studies**: Documentation of expected community benefits
- **Priority Ranking Methodology**: Explanation of how projects were prioritized
